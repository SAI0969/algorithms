Merge Sort Visualization (Divide & Conquer)

Algorithm:

MERGE_SORT(arr, left, right)

1. If left >= right
       return

2. mid = (left + right) / 2

3. MERGE_SORT(arr, left, mid)

4. MERGE_SORT(arr, mid + 1, right)

5. MERGE(arr, left, mid, right)



MERGE(arr, left, mid, right)

1. Create temporary array temp[]

2. i = left
   j = mid + 1
   k = 0

3. While i <= mid AND j <= right
       If arr[i] < arr[j]
            temp[k] = arr[i]
            i++
       Else
            temp[k] = arr[j]
            j++
       k++

4. Copy remaining elements from left half

5. Copy remaining elements from right half

6. Copy temp[] back into arr
