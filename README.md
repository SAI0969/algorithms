# Algorithms Visualization (DAA)

A collection of **interactive algorithm visualizations built using HTML, CSS, and JavaScript** to help students understand **Design and Analysis of Algorithms (DAA)** concepts.

This project provides a simple interface where users can **select algorithms and observe their behavior visually**, making it easier to learn how different algorithms work.

---

## 📌 Overview

Understanding algorithms from textbooks can sometimes be difficult.
This project aims to make algorithm learning **more intuitive and interactive** by providing visual demonstrations of commonly used algorithms.

Each algorithm is implemented as an **individual HTML page**, and they can be accessed through a main interface.

This repository is especially useful for:

* Students studying **Data Structures and Algorithms**
* Beginners learning **algorithm design**
* Anyone interested in **visualizing algorithm behavior**

## ▶ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/SAI0969/algorithms.git
```

2. Navigate to the project folder

```bash
cd algorithms
```

3. Open the main file

Open **index.html** in any web browser.

You can simply double-click the file or open it with:

* Chrome
* Edge
* Firefox
