Quick Sort Visualization (First Element Pivot)

Algorithm:

partition(a, l, r)

p = a[l]
i = l
j = r + 1

while i < j
    increment i until a[i] >= p
    decrement j until a[j] <= p

    if i < j
        swap a[i], a[j]

swap a[l], a[j]
return j


quick_sort(a, l, r)

if l < r
    s = partition(a, l, r)
    quick_sort(a, l, s-1)
    quick_sort(a, s+1, r)
